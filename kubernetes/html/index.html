Kubernetes Dashboard Plugin



Execute to following steps to deploy and use the Kubernetes Dashboard:


  • Connect to the cluster.
  • 1. Prime the cluster.
  • 2. Deploy the Kubernetes Dashboard Helm Chart.
  • 3. Port forward 8443:8443 from the Kubernetes Dashboard POD.
  • 4. Open Kubernetes Dashboard in embedded browser. This also copies the required token to the clipboard.
  • 4.1 Recopy the token to clipboard.
  • 5 Optionally, undeploy the Kubernetes Dashboard Helm Chart.
  • Disconnect from the cluster.
