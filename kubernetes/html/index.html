Kubernetes Dashboard Plugin



Execute to following steps to install Kubernetes Dashboard Helm Chart and use the Kubernetes Dashboard Web UI:


  1. Connect to the cluster.
  2. Prime the cluster. Creates Service Account, Cluster Role Binding and a (token) Secret in kubernetes-dashboard namespace.
  3. Install the Kubernetes Dashboard Helm Chart to deploy Kubernetes Dashboard Web UI.
  4. Port forward 8443:8443 from the Kubernetes Dashboard POD.
  5. Open Kubernetes Dashboard in this tool window. This also copies the required token to the clipboard.
    1. Copy the token to clipboard.
  6. Optionally, undeploy the Kubernetes Dashboard Helm Chart.
  7. When done disconnect from the cluster.
